# Ideas

1. **Topic browser** — Browse episodes by concept (product-market fit, hiring, growth...) using the existing index
2. **Guest explorer** — Find all episodes by a specific guest or company (Airbnb, Stripe, Google, etc.)
3. **Quote/insight extractor** — Surface specific timestamped quotes on a topic across all episodes
4. **Timeline view** — Browse how topics evolved over the podcast's history
5. **Search** — Full-text search across all transcript content
