## Claude Code

- Run /init to create a CLAUDE.md file with context and instructions
  - Conventions, standards, preferences and guidelines
- Extend capabilities with [Skills](https://github.com/anthropics/skills)
- Terminal navigation and shortcuts
  - Deleting current line: Ctr+A and Ctr+K
  - Aborting/Canceling: Ctr+C
  - Clear: Ctr+L or /clear command
- Define permissions with a .claude/settings.json file
  - CRUD
  - Arguments: ls, grep
